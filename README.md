# E-commerce React

Proyecto de e-commerce construido con React, Vite y Tailwind CSS.

## Features
- Catalogo de productos con busqueda y filtros
- Carrito persistente y gestion de cantidades
- Rutas protegidas con React Router
- Estado global con Zustand

## Setup
1. Instalar dependencias:
   ```bash
   npm install
   ```
2. Levantar el entorno de desarrollo:
   ```bash
   npm run dev
   ```

## Deploy (GitHub Pages)
1. Configurar GitHub Pages para publicar desde la rama `gh-pages`.
2. Ejecutar el deploy:
   ```bash
   npm run deploy
   ```
